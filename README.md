Este é um projeto simples de uma página de receita desenvolvido para praticar os fundamentos de HTML5 e CSS3. O objetivo foi criar uma interface limpa, legível e responsiva para exibir ingredientes e instruções de preparo.

🚀 Tecnologias Utilizadas
Para este projeto, utilizei o básico do desenvolvimento web:

HTML5: Estruturação semântica do conteúdo (listas, títulos, seções).

CSS3: Estilização visual (fontes, cores, espaçamentos e bordas).

Google Fonts: Para uma tipografia mais moderna e convidativa.

🛠️ Funcionalidades
Lista de Ingredientes: Organizada com marcadores para facilitar a leitura.

Passo a Passo: Lista numerada detalhando o modo de preparo.
